# Diseño Web - SST-Registro

Prototipo funcional del sistema SST-Registro usando HTML, CSS y JavaScript.

---

## Archivos

| Archivo | Descripción |
|---------|-------------|
| `index.html` | Página principal con todas las pantallas |
| `styles.css` | Estilos CSS del prototipo |
| `script.js` | JavaScript para interactividad |

---

## Cómo Usar

1. Abrir `index.html` en un navegador web
2. Hacer clic en "Ingresar" para simular login
3. Navegar entre las secciones usando el menú lateral

---

## Pantallas Incluidas

- **Login** - Pantalla de inicio de sesión
- **Dashboard** - Vista general con estadísticas y gráficos
- **Registrar** - Formulario para nuevos eventos
- **Consultar** - Lista de eventos con filtros
- **Estadísticas** - Gráficos y reportes
- **Usuarios** - Gestión de usuarios (demo)

---

## Funcionalidades Demo

- Navegación entre páginas
- Formularios interactivos
- Modales (pop-ups) para detalle e investigación
- Gráficos de barras y pastel
- Notificaciones toast
- Tablas con datos de ejemplo

---

## Tecnologías

- HTML5
- CSS3 (con variables CSS)
- JavaScript Vanilla
- Bootstrap 5.3.2 (CDN)
- Bootstrap Icons (CDN)
