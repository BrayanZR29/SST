# SST-Registro

## Sistema de Gestión de Accidentes e Incidentes Laborales

---

## 📋 Descripción

Sistema web para el registro y seguimiento de accidentes e incidentes laborales, diseñado para cumplir con los estándares de Seguridad y Salud en el Trabajo (SST).

---

## 📁 Estructura del Proyecto

```
seguridad_salud_trabajo/
│
├── README.md                    ← Índice principal
│
├── 01-propuesta-software.md     ← Propuesta formal del proyecto
├── 02-diagrama-flujo.md         ← Diagramas de flujo (texto)
├── 03-diseno-software.md        ← Diseño de pantallas (texto)
├── 04-estructura-codigo.md      ← Arquitectura Java/Spring
│
├── uml/                         ← Diagramas UML
│   ├── README.md
│   ├── 01-casos-uso.md          ← Casos de uso (texto)
│   ├── 01-casos-uso.puml         ← Casos de uso (PlantUML)
│   ├── 02-diagrama-clases.md     ← Diagrama de clases (texto)
│   ├── 02-diagrama-clases.puml   ← Diagrama de clases (PlantUML)
│   ├── 03-*.md                   ← Secuencias (texto)
│   ├── 03-secuencia-*.puml       ← Secuencias (PlantUML)
│   ├── 04-actividad-*.puml      ← Actividades (PlantUML)
│   └── 05-diagrama-componentes.puml
│
└── diseno-web/                  ← Prototipo web
    ├── README.md
    ├── index.html               ← Página principal
    ├── styles.css               ← Estilos CSS
    └── script.js                ← JavaScript
```

---

## 🚀 Funcionalidades Principales

- ✅ Registro de accidentes e incidentes
- ✅ Clasificación por tipo y gravedad
- ✅ Investigación de eventos
- ✅ Seguimiento de acciones correctivas
- ✅ Estadísticas y gráficos
- ✅ Generación de reportes PDF
- ✅ Gestión de usuarios con roles
- ✅ Cumplimiento de normativa legal

## 📊 Generar Diagramas PlantUML

Para convertir los archivos `.puml` a imagen:

1. Ir a: https://www.plantuml.com/plantuml/uml/
2. Pegar el código del archivo `.puml`
3. Ver el diagrama generado

O instalar localmente:
```bash
# Ubuntu/Debian
sudo apt install graphviz
plantuml archivo.puml
```
